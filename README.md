# vue_shop

> 一个简单的基于vue的商城前端

## 开发环境

```bash
npm install
npm run serve
```
网络原因，推荐使用国内npm镜像源

```
npm config set registry https://repo.huaweicloud.com/repository/npm/
npm config set sass_binary_site https://repo.huaweicloud.com/node-sass
npm cache clean -f
```
